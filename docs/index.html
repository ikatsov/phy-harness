UR5e + Robotiq 2F-85

Robotic Manipulation Dev Harness

A practical simulation workflow for building and debugging robot manipulation policies with augmented rollout videos, joint logs, VLM transcripts, and task analyzers.

MuJoCo simulation Policy iteration loop Analyzer-driven debugging Video + logs + transcripts

High-level workflow

1) Inputs

Start from policies/impl/<task>/<task>.yaml with task intent and policy wiring.

2) Design & implement

Coding model (or developer) updates policy code, analyzers, and focused tests for the task.

3) Simulate

Run simulation to generate augmented rollout video, joints.csv, metrics, VLM transcript JSON, and custom analyzer outputs.

4) Coding feedback loop

Inspect video frames and logs, tighten analyzers and tests, implement fixes, then repeat from design.

Core artifacts

Simulation outputs

  • artifacts/<task>/rollout.mp4 (augmented 2x2 view)
  • artifacts/<task>/metrics.txt
  • artifacts/<task>/joints.csv with target vs applied controls
  • rollout.vlm_transcript.json when VLM transcriber is enabled
  • Optional task-specific analyzer JSON files

Why this loop works

  • Visual evidence catches control/physics pathologies fast
  • Joint logs make hypotheses testable and repeatable
  • Analyzers convert observations into structured signals
  • Focused tests prevent regressions while iterating quickly
