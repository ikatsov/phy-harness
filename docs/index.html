PhyHarness A Dev Harness for Physical AI

A practical simulation workflow for building and debugging robot manipulation policies with augmented rollout videos, joint logs, VLM transcripts, and task analyzers.

MuJoCo simulation Policy iteration loop Analyzer-driven debugging Video + logs + transcripts IK motion planning
Development loop

High-level workflow

01

Inputs

Define intent in a task YAML spec.
<task>.yaml under
policies/impl/<task>/

02

Design & implement

Coding model implements policy, analyzers, and unit tests for the task.

03

Simulate

Produces rollout.mp4, joints.csv, metrics, VLM transcript, and custom analyzer outputs.

04

Redesign

Inspect video frames, logs, and transcripts. Run ad hoc test and simulation. Design fixes.

Loop back to design & implement
Outputs

Core artifacts

Simulation outputs

  • rollout.mp4 — augmented 4-camera 2×2 grid with kinematic overlays
  • joints.csv — joint qpos + target vs applied controls per step
  • metrics.txt — episode summary
  • rollout.vlm_transcript.json — per-frame VLM descriptions when enabled
  • Custom analyzer JSON files per task

Why this loop works

  • Visual evidence catches control/physics pathologies immediately
  • Joint logs make hypotheses testable and repeatable
  • VLM transcripts describe each sampled frame in natural language
  • Custom analyzers convert observations into structured signals
  • Focused tests prevent regressions during iteration
